# CC项目说明

CC项目旨在构建一套可扩展的IoT设备和工作站集中管理控制系统。

## 项目目标

1. 项目能够实现工作站和IoT的遥测及远程控制；
2. 项目既可以部署在100台设备以内的无服务器的单机房环境，也可以实现10W+ IoT设备的跨域跨云监视和远程控制；
3. 遥测端（也称服务端）容易扩展，实现不同遥测指标的获取，遥测数据具有自说明特性，包括名称、描述、数据类型、范围、采集间隔、数值等信息；
4. 控制端（也称客户端）支持通过插件扩展功能，支持通过扩展市场安装第三方扩展，界面能够完全自定义，可以通过界面市场方便安装第三方设计的界面；
5. 控制命令、脚本可持久化，可模板化，可导入，可收藏，可编辑；
6. 支持对工作站、IoT设备的批量操作。

## 项目前期目标

1. 争对项目目标设计并实现基础架构。
2. 实现对CPU、内存、网络、进程数量和磁盘的遥测。
3. 实现对指定进程的检测。

## 约束

项目以RUST生态为基础构建。
